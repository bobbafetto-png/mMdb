# mMdb

mMdb är ett personligt projekt för att katalogisera och hantera en filmsamling.  
Fokus ligger på enkel översikt, tydlig metadata och ett UI som är snabbt att använda.

Projektet är under aktiv utveckling.

---

## Status
- **Version:** v0.1
- **Branch-struktur:**
  - `main` – stabil baseline
  - `dev` – pågående utveckling och experiment

---

## Teknik
- .NET / .NET MAUI
- C#
- XAML

## Funktionalitet (nuvarande)

## Arbetsflöde
